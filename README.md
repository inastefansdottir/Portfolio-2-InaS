## Description

This is my final portfolio website for the Front-end Development course at Noroff. The portfolio presents selected projects from my studies, including case studies, project reflections, live demos, GitHub repositories, and information about my skills, background, and experience.

The goal of the portfolio was to create a personal and polished website that combines my background in graphic design with my frontend development work. I focused on responsive design, visual identity, interaction details, and clear project presentation.

## Getting Started

### Installing

Clone the repository:

```bash
git clone https://github.com/inastefansdottir/Portfolio-2-InaS.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Deployment

The project is deployed on Netlify.

For React Router refresh support, the project includes a `_redirects` file in the public folder:

```bash
/* /index.html 200
```
